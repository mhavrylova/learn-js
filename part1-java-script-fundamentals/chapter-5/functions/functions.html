Теорія

Функції потрібні для того, щоб не повторювати один і той самий код в багатьох місцях.
Для створення функції, ми використовуємо її оголошення.

function name(parameters) {
body
}

Викликатись ф-ція може таким чином functionName(). Такий виклик виконає код ф-ції

Змінні і функції

Змінні, що були оголошені всередиці ф-ції, видимі тільки в ній, а не за її межами. Якщо до такої змінної звернутись за межами ф-ції, то буде undefined

У функцїї є доступ до зовнішніх змінних і вона може змінювати їх значення. Наприклад, була оголошена змінна, потім ми її в функції переписуємо. Поки функція не буде викликана, то змінна лишиться з початковим значенням, але після виклику ф-ції, змінна вже матиме нове значення, яке ми задали в ф-ції.

Зовнішня змінна використовується тільки якщо всередині ф-ції немає локальної змінної з таким же іменем, яка оголошується в цій ф-ції. Якщо ж одноіменна змінна оголошується всередині функції, то вона перекриває зошнішню. Але лише в межах функції, тобто в ф-ції створюється своя локальна змінна. Якщо ф-ція буде викликана, то вона не перезапише зовнішню (глобальну) змінну.

Параметри та аргументи

В функію можна передати інформацію через параметри: functionName(param1, param2){body with using parameters} і використати ці параметри як змінні всередині функції

Коли ми викликаємо функцію, то ті значення, що передаються в дужках називається агрументами

Якщо при виклику функції агрумент не був переданий, то його значення стає undefined

А можна параметру задати дефолтне значення, яке буде присвоюватись замість undefined, якщо такий агрумент не був переданий functionName(param1, param2 = "value"){body with using parameters} також в параметр можна передати не просто значення, а щось намагато складніше, наприклад функцію, яка повертає якесь значення

Return

Функція може повернути результат, який буде переданий в її виклик.

function sum(a, b) {
return a + b;
}

let result = sum(1, 2);
alert( result ); // 3

Директива return може знаходитись в будь-якому місці функції. Коли виконання доходить до return, функція зупиняється і значення повертається. Якщо використати return без значення, то відбудеться просто вихід із функції. Результат функції буде undefined. Такий же результат ф-ції без return

Якщо після return перервати ентером рядок і вже з наступного рядка писати значення, то код не виконається. Автоматично буде проставлені після return крапка з комою і функція зупинеться
