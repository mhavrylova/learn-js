Теорія

let func = (arg1, arg2, ...argN) => expression;

Створюється функція func, яка приймає аргументи arg1..argN, потім вираховує expression в правій частині з їхнім використанням і повертає результат.
Можна переписати так:

let func = function(arg1, arg2, ...argN) {
return expression;
};

Якщо ми маємо лише один аргумент, то круглі дужки навколо параметрів можна не ставити

Якщо аргументів нема, то круглі дужки будуть пустими, але вони повинні бути

Якщо стрілкова функія складна і містисть багато рядків, то вона повинна бути огорнути фігурними дужками. І для повернення значення цієї функції необхідно використати return

Задача 1

Замініть код Function Expression стрілковою функцією:

function ask(question, yes, no) {
if (confirm(question)) yes()
else no();
}

ask(
"Вы согласны?",
function() { alert("Вы согласились."); },
function() { alert("Вы отменили выполнение."); }
);
