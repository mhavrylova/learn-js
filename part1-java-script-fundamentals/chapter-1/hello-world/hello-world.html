Теорія

Для того, щоб вставити скріпт в будь-яку частину сторінки, використовують тег <script>. В старому коді (HTML, HTML4) можуть зустрічатись застарілі атрибути для цього тегу - type та language.

Великий JS код краще помістити в окремий файл, підключивши його до HTML з використанням атрибуту src <script src="/path/to/script.js"></script> При цьому, в сам тег вже код вписати не можна, він буде ігноруватись.
