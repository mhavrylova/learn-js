type="module" говорить браузеру, що скрипт повинен вважатись як ECMAScript module (ESM), треба для import, export. Використала атрибут defer, щоб скрипти завантажувались паралельно з контентом і відпрацювали одразу після заувантаження контенту. При цьому важливо, щоб скрипти відпрацювали в певному порядку лінкування.
