Задача 2

Уявіть, що у вас є три скрипти, які вам потрібно включити на сторінці, і вони залежать один від одного. Створіть HTML-сторінку з тегами script для цих скриптів так, щоб вони завантажувались та виконувались в правильному порядку.
'constants.js' - має містити перевикористовуємі константи
'action.js' - має містити функцію
'helpers.js' - має містити функції помічники
'main.js' - має містити запуск функціі

Завдання зробити функцію яка приймає строку і валідує її. (action)
придумати самій деклілька правил типо (розмір більше 3х символів, має бути строко, не має включати літеру 'a', тощо.) (helpers)
якщо валідація проходить успішно функція повертає true якщо щось не проходить фолс
в кінці викличи функцію декілько разів (у файлі main) , і після кожного виведи в консоль текст is valid або is invalid (ці строки повині зберігатись в constants)
