Змінні та константи

Змінні

Змінна - це іменоване сховище даних, в якому зберігається якась інформація, потрібна для розробки (наприклад, ім'я юзера, товар в інтернет магазині). Змінну створюють за допомогою ключового слова let (застаріла var). В чому різниця між вар та лет, це найпопулярніше питаня для джунів на інтервью, якщо зараз не зможешь відповісти запамʼятай потім оговоримо. створимо (оголосимо) змінну name та помістимо в неї дані (оголосимо/визначимо змінну) let name; // яке значеня зараз в змінній?
name = 'Marina'; Ми зберегли строку 'Marina' в змінній name. Можна одразу записати ім'я в змінну: let name = 'Marina'; можна записати багато змінних під однією let let user = 'John',
    age = 25,
    message = 'Hello'; // так робити не треба, не дуже явне створеня констант погіршує розуміння коду)

Змінну можна перезаписувати, при цьому попередні значення стираються. let name;
name = 'Marina';
name = 'Dania'; Дані із однієї змінної можна скопіювати в другу змінну. let name = 'Marina';
let person;
person = name;

Змінна оголошується лише 1 раз. Якщо заново оголосити цю саму змінну через let, виникне помилка. Якщо змінну вже оголошено, наступні рази її потрібно використовувати без let

Імена змінних повинні містити лише букви, цифри (але ім'я змінної не може починатись з цифри), символи $ та _.
Якщо в імені змінної декілька слів - можна використати camelCase.
Має значення регіст літер: name i NAME - це різні змінні.
Можна використовувати літери ріхних мов (але це не рекомендовано) // теж не треба можешь це видалити зайва інфа))
Існують зарезервовані слова, які не можна використовувати як назву зміннної

Константи

Константа - змінна, яку не можна змінити/перезаписати. Для її оголошення використовують const. Якщо константу спробувати змінити, виникне помилка.

Задачі

Задача 1

1) Оголосіть дві змінні: admin та name.
2) Присвойте значення "Іван" змінній name.
3) Скопіюйте значення зі змінної name в admin.
4) Виведіть значення змінної admin, використовуючи функцію alert (яка повинна показати “Іван”).
