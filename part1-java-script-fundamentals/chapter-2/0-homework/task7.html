При порівнянні значень за нестрогою рівністю ( == ), такі значення дорівнюють false:

  • false: Логічне значення false завжди дорівнює false.
  • число 0
  • "" (пустий рядок)
  • null
  • undefined
  • NaN
  • [] (пустий масив)
  • {} (пустий об'єкт)
  • function() {} (пуста функція)
  • document.all (особливий об'єкт, вважається "false" у браузерному середовищі.
