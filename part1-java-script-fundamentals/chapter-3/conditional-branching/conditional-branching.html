Теорія

Інструкція if(...) обчислює умову в дужках і перетворює результат у логічний тип, якщо результат умови true, виконує блок коду у фігурних дужках7 if (name == 'JavaScript') {
alert( "good for you" );
}

Вираз if може містити необов’язковий блок else. Він виконується, коли умова false. if (name == 'JavaScript') {
alert( "good for you" );
} else {
alert( "try again" );
}

Щоб перевірити кілька варіантів умови, можна використати блок else if if (name == 'JavaScript') {
alert( "good for you" );
} else if (name == 'TypeScript') {
alert( "not bad" );
} else {
alert( "try again" );
}

Можна умову написати скорочено: let result = ( умова ) ? значенняЯкщоTrue : значенняЯкщоFalse; Для конструкції якщо умов декілька (if else) скорочено використовують декілька ? let message = (age < 3) ? 'Привіт, крихітко!' :
(age < 18) ? 'Вітаю!' :
(age < 100) ? 'Моє шанування!' :
'Який незвичайний вік!';
