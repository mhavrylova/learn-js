Оператори нульового злиття та присвоювання ??

Оператор ?? повертає перший аргумент, якщо він не null/undefined, інакше - другий. a ?? b // якщо а не null/undefined, то a. Інакше - b це те ж саме, що result = (a !== null && a !== undefined) ? a : b;

оператор ?? потрібен, щоб задати значення по замовчуванню для потенційно невизначеної змінної.

різниця || та ?? в тому, що оператор || не відрізняє false, 0, пустий рядок "" і null/undefined. Для нього вони всі однакові, та являються false. Якщо перший аргумент для оператора || буде щось з цих значень, то результатом буде другий аргумент. let height = 0;
alert(height || 100); // 100
alert(height ?? 100); // 0
