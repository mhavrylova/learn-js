Пояснення: в validateString в змінну result записується результат порівняння 3х функцій. Якщо всі вони мають результат true, то повернеться результат останньої функції, він буде true.

Якщо результат будь-якої із трьох функцій буде false, то в result повернеться false

Потім за допомогою switch ми значення цього result порівнюєм в case з true. І в випадку, якщо він true, повертаємо повідомлення про валідність, інакше - про невалідність
