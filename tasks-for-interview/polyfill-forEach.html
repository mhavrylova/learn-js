Написати поліфіл для forEach

arr.forEach(callback)) – вызывает func для каждого элемента. Ничего не возвращает.

Метод forEach() выполняет функцию callback один раз для каждого элемента, находящегося в массиве в порядке возрастания. Она не будет вызвана для удалённых или пропущенных элементов массива. Однако, она будет вызвана для элементов, которые присутствуют в массиве и имеют значение undefined.

Оригінал

var arr = ["Яблоко", "Апельсин", "Груша"];
arr.forEach(function(item, i, arr) {
  alert( i + ": " + item + " (массив:" + arr + ")" );
});

Рішення 1 з прототипом

Array.prototype.myForEach = function (callbackFn) {
  for (let i = 0; i < this.length; i++) {
    callbackFn(this[i], i, this);
  }
}

Рішення 2 з функцією

let createForEach = function (arr, callbackFn){
  for (let i = 0; i < arr.length; i++) {
    callbackFn(arr[i], i, arr);
  }
}

Вивести в консоль всі елементи масиву ["January", "February", "March", "April", "May"], в якому перед кожним місяцем буде стояти його індекс
